
\begin{align*}
    \left\| V^0 - V^* \right\|_\infty &= 3 \\
    \left\| V^1 - V^* \right\|_\infty &= 2 \\
    \left\| V^2 - V^* \right\|_\infty &= 0 \\
    \left\| V^3 - V^* \right\|_\infty &= 0 \\
\end{align*}

We can see that the error of the value iteration is decreasing monotonically in this example.
